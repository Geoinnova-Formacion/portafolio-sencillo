# 🌍 Proyecto 1: Análisis de Cobertura Vegetal

📌 **Descripción**  
Este proyecto evalúa los cambios en la cobertura vegetal en los últimos 10 años usando imágenes satelitales en QGIS.

🛠 **Tecnologías usadas**  
- QGIS
- Sentinel-2
- Python (Raster Analysis)
